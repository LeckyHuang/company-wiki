# Ingest 操作提示词

将以下内容粘贴给 LLM，替换 `[文件路径]` 后执行。

---

```
请按照 SCHEMA.md 的 Ingest 操作规则，处理以下新文件：

文件路径：[文件路径，例如 sources/proposals/001_湛江电信升级规划方案.pdf]

执行步骤：
1. 读取文件内容
2. 提取核心元数据：行业、项目类型、展厅类型、方案阶段、客户层级、方案内容组合、结果（中标/未中标/未知）
3. 提取核心内容：问题分析、解决思路、方案框架大纲、功能模块、政策依据
4. 按 SCHEMA.md 规则更新对应 wiki 页面（若页面不存在则新建）
5. 更新 wiki/index.md
6. 在 wiki/log.md 追加操作记录

请先输出提取到的元数据确认无误后，再执行 wiki 更新。
```

## 批量 Ingest 提示词（第一批10份）

```
请按照 SCHEMA.md 的 Ingest 操作规则，批量处理 sources/proposals/ 目录下的所有新文件。

处理顺序：按文件名序号从小到大。
每处理完一个文件，输出简要摘要后继续下一个。
全部处理完成后，输出汇总：
- 共处理文件数
- 新建 wiki 页面列表
- 更新 wiki 页面列表
- 发现的潜在问题（政策过期、内容矛盾等）
```
