# Lint 操作提示词

将以下内容粘贴给 LLM 执行健康检查。

---

## 完整 Lint

```
请按照 SCHEMA.md 的 Lint 操作规则，对整个 wiki/ 目录执行健康检查。

今天日期：[填入当前日期，例如 2025-08-01]

检查项：
1. wiki/policies/ 中有效期已过的政策 → 标记 status: outdated
2. wiki/credentials/certificates/ 中有效期不足3个月的证书 → 页面顶部加警告
3. 没有被任何其他页面引用的孤立页面
4. 同一功能模块在不同页面描述存在明显矛盾
5. 内容为空或极少（< 100字）的页面

输出格式：
# Lint Report — [日期]
## 过期政策 (N条)
- 页面路径 | 过期时间 | 建议处理
## 临期证书 (N条)
## 孤立页面 (N条)
## 内容矛盾 (N条)
## 空页面 (N条)

输出报告后，询问我是否立即处理某类问题。
```

---

## 快速 Lint（仅检查证书和政策有效期）

```
请检查 wiki/policies/ 和 wiki/credentials/certificates/ 中所有页面的有效期字段。
今天日期：[填入当前日期]
列出已过期或3个月内到期的条目，注明页面路径和到期时间。
```
